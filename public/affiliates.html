How attribution works

There's no cookie, no session window, no "last click within 30 days." Attribution is per tagged call — the code has to ride on the actual request that gets paid.

1. Tag the call

Add your code to any PulseNetwork x402 request, either way:

?ref=your-code

# or

x-referral-code: your-code

2. It settles, you're credited

The moment that specific call settles payment, the revenue, call count, and payer are logged against your code — instantly, on our existing analytics pipeline. No batch jobs, no manual reconciliation on our end.

This means: promotion has to route actual API calls through your code — a link a human clicks and abandons earns nothing, but every real call your integration makes on a user's behalf is credited the moment it settles.
