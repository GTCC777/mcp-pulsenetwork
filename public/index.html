How it works

Every endpoint is gated by x402, the open HTTP-native micropayment standard. An agent calls an endpoint; the server replies 402 Payment Required with a price; the agent pays in USDC on Base or Solana and retries; the endpoint returns the intelligence. Settlement is gasless and on-chain — no invoices, no logins, no rate-limit keys.

Discover

One MCP server exposes all 68 verticals to any agent. Listed on Smithery, Glama, the 402 Index, x402scan and Bazaar.

Pay per query

Cents per call in USDC, settled on Base or Solana. The agent pays only for the answers it actually uses.

Structured intel

Each endpoint synthesizes live public data into decision-ready JSON — global by default, with an openapi.json and llms.txt per vertical.
