gridstatus.io paid tiers / Yes Energy: what you're actually paying for

Yes — rebuildable from free public data

gridstatus.io paid tiers / Yes Energy typically costs hundreds/mo to five figures/yr. Its core dataset is not proprietary: it is built on EIA API; ERCOT / CAISO / ENTSO-E public data — public sources you can access directly for free. What the subscription actually adds is normalized access to grid data the ISOs publish free: LMPs, load, fuel mix.

The price math

Per-lookup at $0.05–0.25 vs subscription tiers.

Licensing and legal traps

  • Some ISO real-time feeds have registration requirements — read each ISO's terms
